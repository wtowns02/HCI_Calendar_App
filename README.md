# HCI Calendar App

A simple, friendly calendar and productivity app for students. More powerful than iPhone Notes, less intimidating than Notion.

![Calendar View](./hci-calendar-app/public/UI_screenshots/Screenshot%202025-11-13%20at%202.53.34%20PM.png)

## Features

**Calendar Views** - Month, week, and day views to organize your schedule.

![Week View](./hci-calendar-app/public/UI_screenshots/Screenshot%202025-11-13%20at%202.53.50%20PM.png)

**To-Do List** - Track tasks with clickable checkboxes.

![To-Do List](./hci-calendar-app/public/UI_screenshots/Screenshot%202025-11-13%20at%202.54.25%20PM.png)

**Reminders** - Set reminders with audio notifications and snooze functionality.

![Reminders](./hci-calendar-app/public/UI_screenshots/Screenshot%202025-11-13%20at%202.54.38%20PM.png)

## Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm run dev
```
