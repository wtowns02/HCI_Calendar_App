# HCI Calendar App

A simple, friendly calendar and productivity app for students. More powerful than iPhone Notes, less intimidating than Notion.

![Calendar Month View](hci-calendar-app/public/UI_screenshots/Calendar_Month_View.png)

## Features

**Calendar Views** - Month, week, and day views to organize your schedule.

![Calendar Week View](hci-calendar-app/public/UI_screenshots/Calendar_Week_View.png)
![Calendar Day View](hci-calendar-app/public/UI_screenshots/Calendar_Day_View.png.png)

**To-Do List** - Track tasks with clickable checkboxes.

![To-Do List](hci-calendar-app/public/UI_screenshots/ToDo_List_View.png)

**Reminders** - Set reminders with audio notifications and snooze functionality.

![Reminders](hci-calendar-app/public/UI_screenshots/Reminder_View.png)
![Add Event Modal](hci-calendar-app/public/UI_screenshots/Add_Event.png)

## Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm run dev
```
